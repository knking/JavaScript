Arrays

Basic Array Methods

  1. pop() - remove element fromend
  2. push() - add element at end
  3. shift() - remove element from start
  4. unshift() - add element at start
  5. concat() - add 2 or more array and return a new array
  6. indexOf() - it will give you index of element
  7. includes() it will verify that element is presenet in that array and true or false
  8. reverse() -it will reverse the array and modify the existing array
  9. sort() -it will sort the array in alphabatically and also modify the original array but if we apply this on number the it will not work properly and it will sort number also on the basisi of alphabate, sort also depend on case sensitive(it work different on small latter and capatoal latter), so we need to use carefully it
  10. slice() - it will slice the element based in index and it will not modify the original array, if we not provide end value then it will go through end
  11. splice() -it will work same as slice but it will remove the element from old array and it modify old array and return a new array
  12. animals.splice(3,2, "new","animal",90) // here value from index 3 will delete upto 2 index(end will be excluded) and new value new animal and 90 will be added in array
